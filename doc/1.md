## 安装Emqtt
- 官方文档: http://emqtt.com/docs/v2/index.html
- 下载  http://emqtt.com/downloads 
- 安装 
```go
unzip emqttd-macosx-vx.x.zip
```
- 常用命令
```
cd emqttd

# 启动emqttd
./bin/emqttd start

# 检查运行状态
./bin/emqttd_ctl status

# 停止emqttd
./bin/emqttd stop



# 3.0版本使用 emqx(把emqttd统一换为emqttd)

```
